pizzaria-nodejs

Criando um sistema de banco de dados e login de uma pizzaria nodejs (iTalents)

## Instalação

1. Baixe todo o código;
2. Abra o terminal do VS Code;
3. Execute: npm i
4. Rode usando: npm run dev
